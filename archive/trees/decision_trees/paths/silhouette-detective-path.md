# Silhouette Detective Path

Use this path when you can only see the tree's silhouette or overall outline, such as from a distance or in low light. This path focuses on the distinctive outline and form of trees.

```
├── SILHOUETTE or OUTLINE is all I can see!
│   ├── What's the OVERALL SHAPE of the silhouette?
│   │   ├── PYRAMIDAL or CONICAL (Christmas tree shape) → CONICAL SILHOUETTE GROUP
│   │   │   ├── PERFECT SYMMETRICAL PYRAMID
│   │   │   │   ├── SPRUCE GENUS (Picea) - Perfect pyramids
│   │   │   │   │   ├── Blue Spruce (P. pungens): Perfect symmetrical pyramid, blue-gray color
│   │   │   │   │   ├── Sitka Spruce (P. sitchensis): Dense conical shape, coastal habitat
│   │   │   │   │   ├── Norway Spruce (P. abies): Classic Christmas tree shape with drooping branches
│   │   │   │   │   ├── All spruce maintain pyramidal shape throughout life
│   │   │   │   │   └── All have branches in distinct tiers around central trunk
│   │   │   │   │
│   │   │   │   ├── FIR GENUS (Abies) - Symmetrical cones
│   │   │   │   │   ├── White Fir (A. concolor): Perfect symmetrical pyramid, silver-blue color
│   │   │   │   │   ├── Red Fir (A. magnifica): Dense symmetrical cone, dark green to blue-green
│   │   │   │   │   ├── Noble Fir (A. procera): Striking symmetrical form with bluish foliage
│   │   │   │   │   ├── All true firs have very symmetrical, dense pyramidal crowns
│   │   │   │   │   └── All maintain perfect conical silhouette more consistently than other conifers
│   │   │   │   │
│   │   │   │   └── YOUNG REDWOOD & SEQUOIA GROUP
│   │   │   │       ├── Young Coast Redwood (Sequoia sempervirens): Narrow conical form when young
│   │   │   │       ├── Young Giant Sequoia (Sequoiadendron giganteum): Conical becoming columnar with age
│   │   │   │       ├── Dawn Redwood (Metasequoia glyptostroboides): Conical with feathery appearance
│   │   │   │       ├── All young redwoods/sequoias show perfect symmetrical pyramidal forms
│   │   │   │       └── All develop more columnar forms with age
│   │   │   │
│   │   │   ├── LESS PERFECT, MORE OPEN PYRAMID
│   │   │   │   ├── PINE GENUS (Pinus) - Open pyramids
│   │   │   │   │   ├── Young Ponderosa Pine (P. ponderosa): Conical when young, opens with age
│   │   │   │   │   ├── Young Jeffrey Pine (P. jeffreyi): Similar to ponderosa but with bluish cast
│   │   │   │   │   ├── Lodgepole Pine (P. contorta): Narrowly conical in dense stands
│   │   │   │   │   ├── All pines typically show conical form when young
│   │   │   │   │   └── All have more open, irregular branch spacing than firs/spruces
│   │   │   │   │
│   │   │   │   ├── DOUGLAS-FIR GENUS (Pseudotsuga)
│   │   │   │   │   ├── Coast Douglas-fir (P. menziesii): Conical with slightly drooping leader
│   │   │   │   │   ├── Rocky Mountain Douglas-fir (P. menziesii var. glauca): Narrower conical form
│   │   │   │   │   ├── All Douglas-firs have softer silhouette than true firs
│   │   │   │   │   └── All show slightly drooping branch tips giving feathery appearance
│   │   │   │   │
│   │   │   │   └── CEDAR & CYPRESS GROUP
│   │   │   │       ├── Incense-cedar (Calocedrus decurrens): Narrow pyramidal form with dense foliage
│   │   │   │       ├── Western Redcedar (Thuja plicata): Pyramidal with drooping branch tips
│   │   │   │       ├── Monterey Cypress (C. macrocarpa): Broadly conical when young
│   │   │   │       ├── All cedar/cypress species form distinctive pyramidal crowns
│   │   │   │       └── All have tapering form with widest point at or near base
│   │   │   │
│   │   │   └── EXTREMELY NARROW PYRAMID/SPIRE
│   │   │       ├── ITALIAN CYPRESS (Cupressus sempervirens)
│   │   │       │   ├── Extremely narrow columnar/spire form
│   │   │       │   ├── Almost pencil-like silhouette
│   │   │       │   └── No spreading branches visible in outline
│   │   │       │
│   │   │       ├── COLUMNAR CULTIVARS
│   │   │       │   ├── 'Skyrocket' Juniper: Extremely narrow columnar form
│   │   │       │   ├── Columnar arborvitae: Pencil-like forms
│   │   │       │   ├── Fastigiate cultivars of many species: Compressed upright forms
│   │   │       │   └── All have width-to-height ratio less than 1:8
│   │   │       │
│   │   │       └── LOMBARDY POPLAR (Populus nigra 'Italica')
│   │   │           ├── Distinctively narrow, columnar deciduous tree
│   │   │           ├── Branches ascending close to trunk
│   │   │           └── Often planted in rows as windbreaks
│   │   │
│   │   ├── COLUMNAR or TALL CYLINDER → COLUMNAR SILHOUETTE GROUP
│   │   │   ├── BROAD COLUMNAR FORMS (width-to-height ratio 1:3 to 1:5)
│   │   │   │   ├── MATURE REDWOOD & SEQUOIA
│   │   │   │   │   ├── Mature Coast Redwood (Sequoia sempervirens): Tall columnar trunk with small crown
│   │   │   │   │   ├── Mature Giant Sequoia (Sequoiadendron giganteum): Massive columnar trunk with rounded crown
│   │   │   │   │   ├── Both create distinctive columnar silhouettes visible from great distances
│   │   │   │   │   └── Both develop more irregular crown outlines with extreme age
│   │   │   │   │
│   │   │   │   ├── EUCALYPTUS GENUS (Eucalyptus)
│   │   │   │   │   ├── Blue Gum Eucalyptus (E. globulus): Tall with upright crown form
│   │   │   │   │   ├── Lemon-scented Gum (Corymbia citriodora): Tall columnar crown
│   │   │   │   │   ├── All eucalyptus tend to grow with strong vertical emphasis
│   │   │   │   │   └── All develop distinctive upright crown silhouettes visible from distance
│   │   │   │   │
│   │   │   │   └── FOREST-GROWN CONIFERS
│   │   │   │       ├── Dense-stand Lodgepole Pine (P. contorta): Slender columnar form in forest
│   │   │   │       ├── Other forest conifers: Develop columnar form due to competition
│   │   │   │       ├── All often form columnar crowns due to forest competition
│   │   │   │       └── All show more columnar form in youth, broadening with age
│   │   │   │
│   │   │   └── NARROW COLUMNAR FORMS (width-to-height ratio less than 1:5)
│   │   │       ├── ITALIAN CYPRESS (Cupressus sempervirens)
│   │   │       │   ├── Extremely narrow, pencil-like column
│   │   │       │   ├── Very compressed width-to-height ratio
│   │   │       │   └── Often planted in groups for dramatic effect
│   │   │       │
│   │   │       ├── LOMBARDY POPLAR (Populus nigra 'Italica')
│   │   │       │   ├── Distinctive narrow deciduous column
│   │   │       │   ├── Branches ascending close to trunk
│   │   │       │   └── Commonly planted in rows
│   │   │       │
│   │   │       └── COLUMNAR CULTIVARS
│   │   │           ├── 'Skyrocket' Juniper (J. scopulorum): Extremely narrow columnar form
│   │   │           ├── 'Blue Arrow' Juniper (J. virginiana): Pencil-thin blue-gray crown
│   │   │           ├── Columnar arborvitae: Narrow pencil-like forms
│   │   │           └── Most are cultivated forms selected for specific crown proportions
│   │   │
│   │   ├── WEEPING or CASCADING → WEEPING SILHOUETTE GROUP
│   │   │   ├── CLASSIC WEEPING FORMS
│   │   │   │   ├── WEEPING WILLOW (Salix babylonica)
│   │   │   │   │   ├── Iconic fountain-like cascade of branches
│   │   │   │   │   ├── Branches rise then dramatically arch downward
│   │   │   │   │   └── Creates rounded crown outline with cascading skirt
│   │   │   │   │
│   │   │   │   ├── WEEPING CHERRY (Prunus species)
│   │   │   │   │   ├── Ornamental trees with strongly drooping branches
│   │   │   │   │   ├── Often grafted on standard (straight trunk)
│   │   │   │   │   └── Create umbrella-like or fountain-like silhouettes
│   │   │   │   │
│   │   │   │   └── OTHER WEEPING ORNAMENTALS
│   │   │   │       ├── Weeping Birch: White bark with pendulous branches
│   │   │   │       ├── Weeping Beech: Dramatic weeping form
│   │   │   │       └── Many are grafted cultivars selected for weeping habit
│   │   │   │
│   │   │   └── SUBTLE WEEPING FORMS
│   │   │       ├── HEMLOCK GENUS (Tsuga)
│   │   │       │   ├── Western Hemlock (T. heterophylla): Conical with drooping leader and branch tips
│   │   │       │   ├── Mountain Hemlock (T. mertensiana): Distinctive drooping branch tips
│   │   │       │   └── All hemlock species have naturally drooping leader (top shoot)
│   │   │       │
│   │   │       ├── CEDAR-LIKE GROUP
│   │   │       │   ├── Western Redcedar (Thuja plicata): Drooping branch tips on conical form
│   │   │       │   ├── Alaska-cedar (Chamaecyparis nootkatensis): Dramatic weeping branchlets
│   │   │       │   └── All maintain overall pyramidal form despite pendulous branch tips
│   │   │       │
│   │   │       └── DROOPING-TIP CONIFERS
│   │   │           ├── Brewer Spruce (Picea breweriana): Distinctive rope-like drooping branches
│   │   │           ├── Norway Spruce (Picea abies): Drooping secondary branches
│   │   │           └── Deodar Cedar (Cedrus deodara): Graceful with drooping branch tips
│   │   │
│   │   ├── ROUNDED or OVAL → ROUNDED SILHOUETTE GROUP
│   │   │   ├── SPHERICAL/BALL-SHAPED CROWNS
│   │   │   │   ├── MAPLE GENUS (Acer)
│   │   │   │   │   ├── Norway Maple (A. platanoides): Dense, symmetrical rounded crown
│   │   │   │   │   ├── Sugar Maple (A. saccharum): Rounded crown in open settings
│   │   │   │   │   ├── All mature maples develop broad, often rounded crown forms
│   │   │   │   │   └── All create distinctive round silhouettes when open-grown
│   │   │   │   │
│   │   │   │   ├── CULTIVATED HORNBEAM/BEECH
│   │   │   │   │   ├── European Hornbeam (Carpinus betulus): Dense rounded crown
│   │   │   │   │   ├── European Beech (Fagus sylvatica): Rounded dense crown
│   │   │   │   │   └── Both create perfect ball-shaped crowns in open settings
│   │   │   │   │
│   │   │   │   └── ORNAMENTAL CHERRIES
│   │   │   │       ├── Yoshino Cherry (Prunus × yedoensis): Rounded vase-shaped crown
│   │   │   │       ├── Kwanzan Cherry (P. serrulata): Rounded crown
│   │   │   │       └── Many form neat rounded outlines, especially when mature
│   │   │   │
│   │   │   ├── OVAL or EGG-SHAPED CROWNS
│   │   │   │   ├── ASH GENUS (Fraxinus)
│   │   │   │   │   ├── Oregon Ash (F. latifolia): Oval crown with upright branches
│   │   │   │   │   ├── White Ash (F. americana): Oval to rounded crown
│   │   │   │   │   └── All create distinctive oval silhouettes wider in middle than at top
│   │   │   │   │
│   │   │   │   ├── SOME OAK SPECIES
│   │   │   │   │   ├── Blue Oak (Q. douglasii): Neat oval shape with light gray branches
│   │   │   │   │   ├── Oregon White Oak (Q. garryana): Dense oval crown
│   │   │   │   │   └── Many oaks form oval crowns when semi-forest grown
│   │   │   │   │
│   │   │   │   └── CAMPHOR TREE (Cinnamomum camphora)
│   │   │   │       ├── Dense oval crown with symmetrical form
│   │   │   │       ├── Distinctive layered branch arrangement
│   │   │   │       └── Creates neat oval silhouette in landscape
│   │   │   │
│   │   │   └── SPREADING, BROAD CROWNS
│   │   │       ├── OAK GENUS (Quercus)
│   │   │       │   ├── Valley Oak (Q. lobata): Massive rounded crown with spreading branches
│   │   │       │   ├── California Black Oak (Q. kelloggii): Round-topped with thick branches
│   │   │       │   ├── Coast Live Oak (Q. agrifolia): Dense, rounded evergreen canopy
│   │   │       │   └── All open-grown oaks develop massive spreading crowns
│   │   │       │
│   │   │       ├── SYCAMORE GENUS (Platanus)
│   │   │       │   ├── California Sycamore (P. racemosa): Broad, irregular spreading crown
│   │   │       │   ├── London Plane Tree (P. × acerifolia): Wide spreading crown
│   │   │       │   └── All have massive spreading crowns with irregular outlines
│   │   │       │
│   │   │       └── MATURE SHADE TREES
│   │   │           ├── American Elm (Ulmus americana): Vase-shaped crown with round top
│   │   │           ├── Bigleaf Maple (Acer macrophyllum): Broad rounded crown
│   │   │           ├── Mature trees in open settings develop widest crowns
│   │   │           └── Most deciduous trees tend toward rounded forms with age
│   │   │
│   │   ├── FLAT-TOPPED or UMBRELLA-LIKE → UMBRELLA SILHOUETTE GROUP
│   │   │   ├── PINE GENUS (Pinus) - Mature specimens
│   │   │   │   ├── OLD GROWTH PINES
│   │   │   │   │   ├── Italian Stone Pine (P. pinea): Perfect umbrella-shaped crown
│   │   │   │   │   ├── Mature Torrey Pine (P. torreyana): Flat-topped umbrella form
│   │   │   │   │   ├── Old Ponderosa Pine (P. ponderosa): Forms umbrella-like crown with age
│   │   │   │   │   ├── All umbrella pines develop flat or gently rounded crown tops
│   │   │   │   │   └── All maintain clear trunks with foliage concentrated at top
│   │   │   │   │
│   │   │   │   └── FLAT-TOPPED PINES
│   │   │   │       ├── Jeffrey Pine (P. jeffreyi): Often develops flat top with age
│   │   │   │       ├── Gray Pine (P. sabiniana): Sometimes develops flattened crown
│   │   │   │       └── Many mature pines flatten out when no longer adding height
│   │   │   │
│   │   │   ├── MATURE JUNIPER (Juniperus)
│   │   │   │   ├── Old Western Juniper (J. occidentalis): Often forms umbrella-like crown
│   │   │   │   ├── Sierra Juniper (J. grandis): Distinctive flat or mushroom-shaped crown
│   │   │   │   ├── All ancient junipers may develop distinctive flat-topped crowns
│   │   │   │   └── All show weathered, twisted branch architecture within crown
│   │   │   │
│   │   │   └── SAVANNA TREE FORMS
│   │   │       ├── Acacia species: Classic savanna umbrella crowns
│   │   │       ├── Mesquite species (Prosopis): Develop umbrella-like crowns
│   │   │       ├── Paloverde species (Parkinsonia): Form broad flat-topped crowns
│   │   │       ├── All savanna tree forms develop distinctive horizontally flattened crowns
│   │   │       └── All maximize shaded area beneath with minimal vertical structure
│   │   │
│   │   ├── VASE-SHAPED (narrow at bottom, wide at top) → VASE SILHOUETTE GROUP
│   │   │   ├── CLASSIC VASE SHAPES
│   │   │   │   ├── ELM GENUS (Ulmus)
│   │   │   │   │   ├── American Elm (U. americana): Classic vase shape with arching branches
│   │   │   │   │   ├── Chinese Elm (U. parvifolia): Smaller vase shape
│   │   │   │   │   ├── All elm species develop distinctive vase-shaped crowns at maturity
│   │   │   │   │   └── All have multiple main branches ascending before arching outward
│   │   │   │   │
│   │   │   │   ├── ZELKOVA GENUS
│   │   │   │   │   ├── Japanese Zelkova (Z. serrata): Vase-shaped like elms
│   │   │   │   │   ├── Often planted as street trees
│   │   │   │   │   └── Create similar silhouette to elms but more compact
│   │   │   │   │
│   │   │   │   └── MATURE BEECH (Fagus)
│   │   │   │       ├── European Beech (F. sylvatica): Often develops vase-like form
│   │   │   │       ├── American Beech (F. grandifolia): Similar vase-like silhouette
│   │   │   │       └── Both create symmetrical outlines with greatest width in upper crown
│   │   │   │
│   │   │   └── MULTI-STEMMED VASE FORMS
│   │   │       ├── MULTI-TRUNK MAPLES
│   │   │       │   ├── Vine Maple (A. circinatum): Low spreading multi-stemmed vase
│   │   │       │   ├── Japanese Maple (A. palmatum): Often multi-stemmed vase shape
│   │   │       │   └── Multiple stems create fountain-like or vase-like outline
│   │   │       │
│   │   │       ├── CRAPE MYRTLE (Lagerstroemia)
│   │   │       │   ├── Multiple stems creating vase-like crown
│   │   │       │   ├── Open branch structure clearly visible in silhouette
│   │   │       │   └── Creates distinctive spreading vase shape at maturity
│   │   │       │
│   │   │       └── SERVICEBERRY (Amelanchier)
│   │   │           ├── Western Serviceberry (A. alnifolia): Multi-stemmed vase form
│   │   │           ├── Arching branches creating fountain-like outline
│   │   │           └── All form vase-shaped or fountain-like crowns from multiple stems
│   │   │
│   │   ├── IRREGULAR or SCULPTURAL → IRREGULAR SILHOUETTE GROUP
│   │   │   ├── WINDSWEPT TREE FORMS
│   │   │   │   ├── COASTAL WIND-PRUNED TREES
│   │   │   │   │   ├── Monterey Cypress (C. macrocarpa): Classic flag-form wind-pruned crown
│   │   │   │   │   ├── Shore Pine (P. contorta): Dramatically wind-sculpted crown
│   │   │   │   │   ├── Coast Live Oak (Q. agrifolia): Often wind-shaped along coast
│   │   │   │   │   ├── All coastal trees may develop asymmetrical crown forms from wind exposure
│   │   │   │   │   └── All show distinctive lean away from prevailing wind
│   │   │   │   │
│   │   │   │   └── ALPINE WIND-SCULPTED TREES
│   │   │   │       ├── Whitebark Pine (P. albicaulis): Krummholz or mat-like form at treeline
│   │   │   │       ├── Bristlecone Pine (P. longaeva): Twisted, partially dead crown
│   │   │   │       ├── High-elevation junipers: Sculpted by wind and snow
│   │   │   │       └── All show dramatic sculpting from harsh conditions
│   │   │   │
│   │   │   ├── ANCIENT or GNARLED FORMS
│   │   │   │   ├── ANCIENT OAKS
│   │   │   │   │   ├── Very old Valley Oak (Q. lobata): Gnarled limbs, some dead branches
│   │   │   │   │   ├── Ancient Coast Live Oak (Q. agrifolia): Massive low spreading branches
│   │   │   │   │   └── All ancient oaks develop highly individual, irregular forms
│   │   │   │   │
│   │   │   │   ├── OLD GROWTH CONIFERS
│   │   │   │   │   ├── Ancient redwoods/sequoias: Broken tops, massive irregular crowns
│   │   │   │   │   ├── Old growth pines: Lightning damage, asymmetrical crowns
│   │   │   │   │   └── All show evidence of storm damage, lightning strikes, etc.
│   │   │   │   │
│   │   │   │   └── DESERT TREE FORMS
│   │   │   │       ├── Joshua Tree (Yucca brevifolia): Bizarre branching pattern
│   │   │   │       ├── Desert ironwood: Twisting, irregular crown
│   │   │   │       └── All develop highly individualized forms in harsh conditions
│   │   │   │
│   │   │   └── PRUNED or HUMAN-INFLUENCED FORMS
│   │   │       ├── POLLARDED TREES
│   │   │       │   ├── Pollarded planes/sycamores: Knobby branches with seasonal regrowth
│   │   │       │   ├── Pollarded mulberries: Ball-like growth on bare framework
│   │   │       │   └── All show distinctive knuckle-like branch ends
│   │   │       │
│   │   │       ├── TOPIARY FORMS
│   │   │       │   ├── Various species pruned into geometric or animal shapes
│   │   │       │   ├── Creates highly artificial silhouettes
│   │   │       │   └── Common in formal gardens and some landscapes
│   │   │       │
│   │   │       └── BONSAI-INFLUENCED LANDSCAPE TREES
│   │   │           ├── Japanese garden pines: Trained into horizontal layers
│   │   │           ├── Cloud-pruned specimens: Creating cloud-like pads of foliage
│   │   │           └── All show deliberate human manipulation of natural growth pattern
│   │   │
│   │   └── PALM or YUCCA-LIKE → PALM SILHOUETTE GROUP
│   │       ├── TRUE PALMS (Arecaceae family)
│   │       │   ├── California Fan Palm (Washingtonia filifera): Round crown of large fan fronds
│   │       │   ├── Mexican Fan Palm (W. robusta): More slender with smaller crown
│   │       │   ├── Date Palm (Phoenix dactylifera): Dense crown of feather-like fronds
│   │       │   ├── All have distinctive unbranched single trunk
```

## Silhouette Identification Tips

* **Distance Matters**: This path works best when you're too far to see detailed features like leaves but can see the entire tree.

* **Seasonal Changes**: Remember that deciduous trees look dramatically different in winter (when bare) compared to summer (when leafy).

* **Compare Proportions**: Notice the tree's height-to-width ratio. Is it much taller than wide (columnar)? Wider than tall (spreading)? This can be a key identification clue.

* **Growth Environment**: Trees growing in forests often have different shapes than the same species growing in open areas.

* **Unusual Trees**: Some trees have such distinctive shapes (weeping willows, Italian cypress, Joshua trees) that they can be identified by silhouette alone.
