# Bark Path

Look at the bark of your tree.

**Special Feature Pages:**
- [Redwood & Sequoia Group](groups/redwood-sequoia-group.md) - Compare the distinctive reddish, spongy bark of redwoods and sequoias
- [White & Peeling Bark Group](groups/white-peeling-bark-group.md) - Identify trees with striking white or peeling bark
- [Red & Orange Bark Group](groups/red-orange-bark-group.md) - Discover trees with warm-colored, eye-catching bark

**Helpful Reference:** Learn about different bark types with our [Bark Features Guide](references/bark-features-guide.md)!

```

│   ├── What's the main COLOR of the bark?
│   │   ├── I'M NOT SURE → Here's a tip:
│   │   │   ├── Look at the trunk in sunlight!
│   │   │   ├── Light colors will appear white, cream, or gray

│   │   │   └── If colors vary, look at the most common color on the main trunk

│   │   │   │   │       ├── White Ironbark: Smooth white bark becoming rough with age

│   │   │   │   │   │   ├── All have distinctive "eye" markings or horizontal scars on trunk
│   │   │   │   │   │   └── All produce small round/triangular leaves that tremble in slight breeze
│   │   │   │   │   │
│   │   │   │   │   ├── ALDER GENUS (Alnus)
│   │   │   │   │   │   ├── White Alder: Light gray bark with horizontal lenticels
│   │   │   │   │   │   ├── Red Alder: Smooth gray bark often with white patches
│   │   │   │   │   │   ├── Green Alder: Smooth grayish bark with scattered lenticels
│   │   │   │   │   │   ├── European Alder: Gray-brown bark with small lenticels
│   │   │   │   │   │   ├── Italian Alder: Smooth gray bark developing small plates with age
│   │   │   │   │   │   ├── All have smooth bark with distinctive lenticel marking pattern

│   │   │   │       │   ├── Western White Pine: Smooth gray bark becoming plated

│   │   │   │       │   ├── Limber Pine: Light gray to whitish smooth bark

│   │   │   │       │   ├── Older specimens develop increasingly rough texture with maturity
│   │   │   │       │   └── All produce heart-shaped to triangular leaves and fluffy cotton-like seeds

│   │   │   │           ├── Blue Oak: Light gray, somewhat checked bark
│   │   │   │           ├── Oregon White Oak: Light grayish bark with shallow fissures
│   │   │   │           ├── Englemann Oak: Grayish bark with blocky plates
│   │   │   │           ├── Bur Oak: Light gray deeply furrowed bark
│   │   │   │           ├── White Oak: Light ashy-gray bark with shallow furrows
│   │   │   │           └── All oak species produce acorns (key oak trait)
│   │   │   │
│   │   │   ├── Does the light bark have a SPECIAL FEATURE?
│   │   │   │   ├── PUZZLE-PIECE PATTERN → PUZZLE BARK GROUP
│   │   │   │   │   ├── SYCAMORE GENUS (Platanus)
│   │   │   │   │   ├── California Sycamore: Classic puzzle-piece patterned bark
│   │   │   │   │   ├── London Plane Tree: Similar puzzle-like pattern (ornamental)
│   │   │   │   │   ├── American Sycamore: White bark with distinct puzzle pattern
│   │   │   │   │   ├── Arizona Sycamore: Similar pattern with more beige tones
│   │   │   │   │   └── All sycamores shed puzzle pieces revealing lighter patches beneath
│   │   │   │   │
│   │   │   │   ├── HORIZONTAL LINES or LENTICELS → LINED BARK GROUP
│   │   │   │   │   ├── BIRCH GENUS (Betula)
│   │   │   │   │   ├── All birch species have distinctive horizontal dash-like lenticels
│   │   │   │   │   ├── Includes Paper Birch, River Birch, Water Birch, Gray Birch
│   │   │   │   │   ├── ALDER GENUS (Alnus)
│   │   │   │   │   ├── All alder species have similar but less prominent horizontal lines
│   │   │   │   │   ├── Includes White Alder, Red Alder, European Alder, Green Alder
│   │   │   │   │   ├── CHERRY GENUS (Prunus)
│   │   │   │   │   └── Many cherry species have distinctive horizontal lenticel lines
│   │   │   │   │
│   │   │   │   └── POWDERY or DUSTY APPEARANCE → POWDERY BARK GROUP
│   │   │   │       ├── EUCALYPTUS GENUS (Eucalyptus)
│   │   │   │       ├── Ghost Gum and several other species: Powdery white bloom on bark
│   │   │   │       ├── Snow Gum: Chalky white appearance with colorful patches
│   │   │   │       ├── PINE GENUS (Pinus)
│   │   │   │       ├── Several pine species develop whitish powdery patches
│   │   │   │       ├── OTHER POWDERY-BARKED TREES
│   │   │   │       ├── Aspen species often have powdery bloom when young
│   │   │   │       └── Basswood: Develops grayish powdery appearance with age
│   │   │   │
│   │   │   └── Where is the tree GROWING?
│   │   │       ├── ALONG STREAMS/WATER → RIPARIAN LIGHT BARK GROUP
│   │   │       │   ├── ASPEN GENUS (Populus)
│   │   │       │   │   ├── Quaking Aspen: Forms groves along streams
│   │   │       │   │   ├── Bigtooth Aspen: Similar white bark, larger teeth on leaves
│   │   │       │   │   ├── European Aspen: Greenish-white bark with similar form
│   │   │       │   │   ├── All have white to greenish-white smooth bark with black markings
│   │   │       │   │   └── All produce round to triangular leaves that "quake" in the breeze

│   │   │       │   ├── ALDER GENUS (Alnus)

│   │   │       │   │   ├── Red Alder: Can have light patches, similar habitat
│   │   │       │   │   ├── Arizona Alder: Gray bark with lenticels, streamside species

│   │   │       │   │   ├── European Alder: Grayish bark with horizontal markings

│   │   │       │       ├── Eastern Cottonwood: Light gray bark becoming furrowed

│   │   │       │       ├── Narrowleaf Cottonwood: Light gray bark with shallow furrows
│   │   │       │       └── All cottonwood species produce fluffy cotton-like seeds in spring

│   │   │           │   ├── Paper Birch: Brilliant white peeling bark

│   │   │           └── CRAPE MYRTLE (Lagerstroemia)

│   │   │               └── Distinctive peeling pattern revealing lighter layers
│   │   │
```
