# Bark Path

Look at the bark of your tree.

```
├── The BARK looks interesting!
│   ├── What's the main COLOR of the bark?
│   │   ├── WHITE, CREAM, or LIGHT GRAY → LIGHT-COLORED BARK GROUP
│   │   │   ├── How does the bark TEXTURE appear?
│   │   │   │   ├── SMOOTH with peeling layers → PEELING LIGHT BARK GROUP
│   │   │   │   │   ├── BIRCH GENUS (Betula)
│   │   │   │   │   │   ├── Water Birch: Smooth reddish-brown with white lenticels
│   │   │   │   │   │   ├── Paper Birch: Brilliant white peeling bark (ornamental)
│   │   │   │   │   │   ├── All have distinctive horizontal lenticels (dash marks)
│   │   │   │   │   │   └── All produce catkins in spring, small winged seeds
│   │   │   │   │   │
│   │   │   │   │   ├── SYCAMORE GENUS (Platanus)
│   │   │   │   │   │   ├── California Sycamore: Mottled white/tan/green peeling bark
│   │   │   │   │   │   ├── Distinctive patchwork or "camouflage" pattern
│   │   │   │   │   │   └── Produces ball-like fruits hanging on strings
│   │   │   │   │   │
│   │   │   │   │   └── SOME EUCALYPTUS SPECIES
│   │   │   │   │       ├── Ghost Gum: Smooth white bark
│   │   │   │   │       ├── Sugar Gum: White to cream bark that sheds
│   │   │   │   │       └── All have distinctive eucalyptus smell when leaves crushed
│   │   │   │   │
│   │   │   │   ├── SMOOTH with distinct markings → MARKED LIGHT BARK GROUP
│   │   │   │   │   ├── ASPEN GENUS (Populus)
│   │   │   │   │   │   ├── Quaking Aspen: Smooth white bark with black knots/scars
│   │   │   │   │   │   ├── Distinctive "eye" markings on the trunk
│   │   │   │   │   │   └── Small round leaves that tremble in slight breeze
│   │   │   │   │   │
│   │   │   │   │   ├── SOME ALDER SPECIES (Alnus)
│   │   │   │   │   │   ├── White Alder: Light gray bark with horizontal lenticels
│   │   │   │   │   │   ├── Smooth bark with distinctive marking pattern
│   │   │   │   │   │   └── Produces woody "cone-like" fruits, catkins in spring
│   │   │   │   │   │
│   │   │   │   │   └── SOME MAPLE SPECIES (Acer)
│   │   │   │   │       ├── Bigleaf Maple: Young trees have light gray smooth bark
│   │   │   │   │       ├── Vine Maple: Smooth greenish to light gray bark
│   │   │   │   │       └── All have opposite branching pattern (key maple trait)
│   │   │   │   │
│   │   │   │   └── ROUGH or BLOCKY texture → ROUGH LIGHT BARK GROUP
│   │   │   │       ├── PINE GENUS - Some species
│   │   │   │       │   ├── Ponderosa Pine: Upper bark yellow/orange in plates
│   │   │   │       │   ├── Sugar Pine: Light gray to whitish plates when mature
│   │   │   │       │   └── Both recognized by needle bundles (key pine trait)
│   │   │   │       │
│   │   │   │       ├── POPLAR GENUS (Populus) - Older specimens
│   │   │   │       │   ├── Cottonwood: Deeply furrowed light grayish bark with age
│   │   │   │       │   ├── Develops increasingly rough texture with maturity
│   │   │   │       │   └── Heart-shaped to triangular leaves, fluffy seeds
│   │   │   │       │
│   │   │   │       └── SOME OAK SPECIES (Quercus)
│   │   │   │           ├── Valley Oak: Light gray, blocky texture with age
│   │   │   │           ├── Blue Oak: Light gray, somewhat checked bark
│   │   │   │           └── Both produce acorns (key oak trait)
│   │   │   │
│   │   │   ├── Does the light bark have a SPECIAL FEATURE?
│   │   │   │   ├── PUZZLE-PIECE PATTERN → PUZZLE BARK GROUP
│   │   │   │   │   ├── California Sycamore: Classic puzzle-piece patterned bark
│   │   │   │   │   ├── Plane Tree: Similar puzzle-like pattern (ornamental relative)
│   │   │   │   │   └── Both shed puzzle pieces revealing lighter patches beneath
│   │   │   │   │
│   │   │   │   ├── HORIZONTAL LINES or LENTICELS → LINED BARK GROUP
│   │   │   │   │   ├── Birch species: Distinctive horizontal dash-like marks
│   │   │   │   │   ├── Alder species: Similar horizontal lines
│   │   │   │   │   └── Cherry species: Some have horizontal lines (ornamental)
│   │   │   │   │
│   │   │   │   └── POWDERY or DUSTY APPEARANCE → POWDERY BARK GROUP
│   │   │   │       ├── Some Eucalyptus: Powdery white bloom on bark
│   │   │   │       ├── Ghost Gum: Chalky white appearance 
│   │   │   │       └── Some pines: Whitish powdery patches
│   │   │   │
│   │   │   └── Where is the tree GROWING?
│   │   │       ├── ALONG STREAMS/WATER → RIPARIAN LIGHT BARK GROUP
│   │   │       │   ├── ASPEN GENUS (Populus)
│   │   │       │   │   ├── Quaking Aspen: Forms groves along streams
│   │   │       │   │   ├── White smooth bark with black markings
│   │   │       │   │   └── Round leaves that "quake" in the breeze
│   │   │       │   │
│   │   │       │   ├── ALDER GENUS (Alnus)
│   │   │       │   │   ├── White Alder: Light gray bark, grows along streams
│   │   │       │   │   ├── Red Alder: Can have light patches, similar habitat
│   │   │       │   │   └── Both have distinctive catkins and cone-like fruits
│   │   │       │   │
│   │   │       │   └── COTTONWOOD GENUS (Populus)
│   │   │       │       ├── Fremont Cottonwood: Light gray bark, riparian species
│   │   │       │       ├── Black Cottonwood: Sometimes has light patches
│   │   │       │       └── Both produce cotton-like seeds in spring
│   │   │       │
│   │   │       ├── IN MOUNTAIN FORESTS → MOUNTAIN LIGHT BARK GROUP
│   │   │       │   ├── ASPEN GENUS (Populus)
│   │   │       │   │   ├── Quaking Aspen: Forms beautiful mountain groves
│   │   │       │   │   ├── White bark stands out against dark conifers
│   │   │       │   │   └── Brilliant yellow fall color
│   │   │       │   │
│   │   │       │   ├── PINE GENUS - Some species
│   │   │       │   │   ├── Whitebark Pine: Light colored bark at high elevations
│   │   │       │   │   ├── Limber Pine: Light gray to whitish bark
│   │   │       │   │   └── Both high-elevation pines with 5 needles per bundle
│   │   │       │   │
│   │   │       │   └── SOME FIR SPECIES (Abies)
│   │   │       │       ├── White Fir: Smooth light gray bark when young
│   │   │       │       ├── Subalpine Fir: Light-colored, smooth bark
│   │   │       │       └── Both have flat needles, upright cones
│   │   │       │
│   │   │       └── IN URBAN SETTINGS → ORNAMENTAL LIGHT BARK GROUP
│   │   │           ├── BIRCH SPECIES - Ornamental plantings
│   │   │           │   ├── Paper Birch: Brilliant white peeling bark
│   │   │           │   ├── River Birch: Peeling cinnamon to cream bark
│   │   │           │   └── Both commonly planted for ornamental value
│   │   │           │
│   │   │           ├── BEECH GENUS (Fagus)
│   │   │           │   ├── European Beech: Smooth light gray bark
│   │   │           │   ├── American Beech: Similar smooth gray bark (rare in CA)
│   │   │           │   └── Both have smooth elephant-like gray bark
│   │   │           │
│   │   │           └── CRAPE MYRTLE (Lagerstroemia)
│   │   │               ├── Crape Myrtle: Smooth mottled bark, light colored
│   │   │               ├── Popular ornamental with colorful flowers
│   │   │               └── Distinctive peeling pattern revealing lighter layers
│   │   │
│   │   ├── RED, ORANGE, or CINNAMON → RED-TONED BARK GROUP
│   │   │   ├── How does the bark TEXTURE appear?
│   │   │   │   ├── SMOOTH and PEELING → PEELING RED BARK GROUP
│   │   │   │   │   ├── MADRONE GENUS (Arbutus)
│   │   │   │   │   │   ├── Pacific Madrone: Smooth peeling red-orange bark
│   │   │   │   │   │   ├── Peels to reveal bright green new bark underneath
│   │   │   │   │   │   └── Leathery oval leaves, produces red berries
│   │   │   │   │   │
│   │   │   │   │   ├── MANZANITA GENUS (Arctostaphylos)
│   │   │   │   │   │   ├── Various manzanita species: Smooth red bark
│   │   │   │   │   │   ├── Similar to madrone but typically smaller
│   │   │   │   │   │   └── Small bell-shaped flowers, berry-like fruits
│   │   │   │   │   │
│   │   │   │   │   └── SOME EUCALYPTUS SPECIES
│   │   │   │   │       ├── Red Gum: Peeling bark revealing red-orange patches
│   │   │   │   │       ├── Other eucalyptus species with red tones
│   │   │   │   │       └── All have distinctive eucalyptus smell when leaves crushed
│   │   │   │   │
│   │   │   │   ├── FIBROUS or SHREDDY → FIBROUS RED BARK GROUP
│   │   │   │   │   ├── REDWOOD GENUS (Sequoia)
│   │   │   │   │   │   ├── Coast Redwood: Reddish-brown fibrous bark
│   │   │   │   │   │   ├── Thick, spongy texture, fire resistant
│   │   │   │   │   │   └── Extremely tall with small flat needles
│   │   │   │   │   │
│   │   │   │   │   ├── SEQUOIA GENUS (Sequoiadendron)
│   │   │   │   │   │   ├── Giant Sequoia: Cinnamon-red, deeply furrowed bark
│   │   │   │   │   │   ├── Very thick, spongy fire resistant texture
│   │   │   │   │   │   └── Massive trunks, small scale-like needles
│   │   │   │   │   │
│   │   │   │   │   └── INCENSE-CEDAR (Calocedrus)
│   │   │   │   │       ├── Incense-cedar: Reddish brown shreddy bark
│   │   │   │   │       ├── Fibrous texture, strongly aromatic
│   │   │   │   │       └── Flat sprays of scale-like foliage
│   │   │   │   │
│   │   │   │   └── PLATED or SCALY → PLATED RED BARK GROUP
│   │   │   │       ├── PINE GENUS - Some species
│   │   │   │       │   ├── Ponderosa Pine: Orange-brown "puzzle piece" plates
│   │   │   │       │   ├── Jeffrey Pine: Similar but larger plates
│   │   │   │       │   └── Both smell like vanilla or butterscotch when warm
│   │   │   │       │
│   │   │   │       ├── JUNIPER GENUS (Juniperus)
│   │   │   │       │   ├── Western Juniper: Reddish brown shredding bark
│   │   │   │       │   ├── California Juniper: Similar color and texture
│   │   │   │       │   └── Both have scale-like foliage, blue berry-like cones
│   │   │   │       │
│   │   │   │       └── SOME PINE SPECIES
│   │   │   │           ├── Torrey Pine: Reddish plated bark
│   │   │   │           ├── Monterey Pine: Reddish-brown furrowed bark
│   │   │   │           └── Both coastal pines with distinctive forms
│   │   │   │
│   │   │   ├── Does the red bark have a SPECIAL FEATURE?
│   │   │   │   ├── GLOSSY, POLISHED look → POLISHED RED BARK GROUP
│   │   │   │   │   ├── Madrone: Glossy, polished-looking red bark
│   │   │   │   │   ├── Manzanita: Similar shiny appearance
│   │   │   │   │   └── Both have smooth texture that feels cool to touch
│   │   │   │   │
│   │   │   │   ├── "SUNBURN" PEELING appearance → PEELING RED BARK GROUP
│   │   │   │   │   ├── Pacific Madrone: Classic "sunburned" peeling appearance
│   │   │   │   │   ├── Some manzanita species: Similar peeling character
│   │   │   │   │   └── Both reveal bright green or yellow new bark underneath
│   │   │   │   │
│   │   │   │   └── CINNAMON "PUZZLE PIECES" → PUZZLE RED BARK GROUP
│   │   │   │       ├── Ponderosa Pine: Classic cinnamon puzzle-piece bark
│   │   │   │       ├── Jeffrey Pine: Similar but larger plates
│   │   │   │       └── Both have distinctive vanilla/butterscotch scent
│   │   │   │
│   │   │   └── How does the tree SMELL when bark is scratched?
│   │   │       ├── VANILLA or BUTTERSCOTCH scent → AROMATIC RED BARK GROUP
│   │   │       │   ├── Ponderosa Pine: Strong vanilla scent in plates
│   │   │       │   ├── Jeffrey Pine: Even stronger butterscotch smell
│   │   │       │   └── Both most aromatic on warm sunny days
│   │   │       │
│   │   │       ├── CEDAR or PENCIL-SHAVING smell → CEDAR RED BARK GROUP
│   │   │       │   ├── Incense-cedar: Strong pencil shaving smell
│   │   │       │   ├── Western Redcedar: Aromatic cedar smell
│   │   │       │   └── Both have fibrous reddish bark
│   │   │       │
│   │   │       └── NO DISTINCT SMELL → NON-AROMATIC RED BARK GROUP
│   │   │           ├── Pacific Madrone: No strong smell when scratched
│   │   │           ├── Coast Redwood: Minimal scent in bark
│   │   │           └── Manzanita species: Typically not aromatic
│   │   │
│   │   ├── DARK BROWN to BLACK → DARK BARK GROUP
│   │   │   ├── How does the bark TEXTURE appear?
│   │   │   │   ├── DEEPLY FURROWED with ridges → RIDGED DARK BARK GROUP
│   │   │   │   │   ├── OAK GENUS - Many species
│   │   │   │   │   │   ├── California Black Oak: Dark deeply furrowed bark
│   │   │   │   │   │   ├── Canyon Live Oak: Very dark, deeply grooved
│   │   │   │   │   │   └── Both produce acorns (key oak trait)
│   │   │   │   │   │
│   │   │   │   │   ├── ASH GENUS (Fraxinus)
│   │   │   │   │   │   ├── Oregon Ash: Dark bark with diamond-shaped furrows
│   │   │   │   │   │   ├── Distinctive interlacing ridge pattern
│   │   │   │   │   │   └── Compound leaves, opposite branching
│   │   │   │   │   │
│   │   │   │   │   └── WALNUT GENUS (Juglans)
│   │   │   │   │       ├── California Black Walnut: Dark deeply furrowed bark
│   │   │   │   │       ├── Northern California Walnut: Similar bark texture
│   │   │   │   │       └── Both have compound leaves, produce walnuts
│   │   │   │   │
│   │   │   │   ├── BLOCKY or CHECKERED → CHECKERED DARK BARK GROUP
│   │   │   │   │   ├── PERSIMMON GENUS (Diospyros)
│   │   │   │   │   │   ├── Common Persimmon: Distinctive checkered bark
│   │   │   │   │   │   ├── Black blocks with lighter cracks between
│   │   │   │   │   │   └── Produces orange persimmon fruits
│   │   │   │   │   │
│   │   │   │   │   ├── DOGWOOD GENUS (Cornus)
│   │   │   │   │   │   ├── Pacific Dogwood: Dark bark breaks into small blocks
│   │   │   │   │   │   ├── Distinctive alligator-like pattern when mature
│   │   │   │   │   │   └── Has large white "flowers" (actually bracts)
│   │   │   │   │   │
│   │   │   │   │   └── OLDER OAK SPECIES
│   │   │   │   │       ├── Valley Oak: Develops checkered pattern with age
│   │   │   │   │       ├── Blue Oak: Sometimes forms small blocks
│   │   │   │   │       └── Both produce acorns (key oak trait)
│   │   │   │   │
│   │   │   │   └── SCALY or FLAKY → SCALY DARK BARK GROUP
│   │   │   │       ├── PINE GENUS - Some species
│   │   │   │       │   ├── Bishop Pine: Dark scaly bark
│   │   │   │       │   ├── Knobcone Pine: Dark flaky bark
│   │   │   │       │   └── Both recognized by needle bundles (key pine trait)
│   │   │   │       │
│   │   │   │       ├── DOUGLAS-FIR GENUS (Pseudotsuga)
│   │   │   │       │   ├── Douglas-fir: Dark bark with resin blisters when young
│   │   │   │       │   ├── Develops thick corky bark with age
│   │   │   │       │   └── Has distinctive cones with "mouse tail" bracts
│   │   │   │       │
│   │   │   │       └── BUCKEYE GENUS (Aesculus)
│   │   │   │           ├── California Buckeye: Dark scaly bark
│   │   │   │           ├── Distinctive rounded crown, early leaf drop
│   │   │   │           └── Produces large, poisonous "horse chestnut" seeds
│   │   │   │
│   │   │   ├── Does the dark bark have a SPECIAL FEATURE?
│   │   │   │   ├── OOZING SAP or RESIN → RESINOUS DARK BARK GROUP
│   │   │   │   │   ├── CHERRY/PLUM GENUS (Prunus)
│   │   │   │   │   │   ├── Various cherry species: Often have resin or gum
│   │   │   │   │   │   ├── Distinctive horizontal lenticels
│   │   │   │   │   │   └── Often among first trees to flower in spring
│   │   │   │   │   │
│   │   │   │   │   ├── SOME PINE SPECIES
│   │   │   │   │   │   ├── Gray Pine: Often has resin on trunk
│   │   │   │   │   │   ├── Bishop Pine: Can have resinous patches
│   │   │   │   │   │   └── Both recognized by needle bundles (key pine trait)
│   │   │   │   │   │
│   │   │   │   │   └── FIR GENUS (Abies) - Young trees
│   │   │   │   │       ├── White Fir: Resin blisters in young bark
│   │   │   │   │       ├── Red Fir: Similar resin features
│   │   │   │   │       └── Both have flat needles, upright cones
│   │   │   │   │
│   │   │   │   ├── WARTY or BUMPY texture → WARTY DARK BARK GROUP
│   │   │   │   │   ├── HACKBERRY GENUS (Celtis)
│   │   │   │   │   │   ├── Western Hackberry: Distinctive warty bark
│   │   │   │   │   │   ├── Numerous warty projections on trunk
│   │   │   │   │   │   └── Produces small berry-like drupes
│   │   │   │   │   │
│   │   │   │   │   ├── SOME MATURE COTTONWOODS (Populus)
│   │   │   │   │   │   ├── Fremont Cottonwood: Develops warty ridges with age
│   │   │   │   │   │   ├── Black Cottonwood: Similar texture when old
│   │   │   │   │   │   └── Both produce cotton-like seeds in spring
│   │   │   │   │   │
│   │   │   │   │   └── BUCKEYE GENUS (Aesculus)
│   │   │   │   │       ├── California Buckeye: Sometimes has warty bark
│   │   │   │   │       ├── Especially noticeable on older specimens
│   │   │   │   │       └── Produces distinctive large round seeds
│   │   │   │   │
│   │   │   │   └── Very THICK and CORKY → CORKY DARK BARK GROUP
│   │   │   │       ├── CORK OAK (Quercus suber)
│   │   │   │       │   ├── Cork Oak: Extremely thick corky bark
│   │   │   │       │   ├── Planted ornamentally, source of commercial cork
│   │   │   │       │   └── Evergreen oak with small acorns
│   │   │   │       │
│   │   │   │       ├── OLDER DOUGLAS-FIR (Pseudotsuga)
│   │   │   │       │   ├── Douglas-fir: Develops very thick corky bark
│   │   │   │       │   ├── Can be up to 12 inches thick on old trees
│   │   │   │       │   └── Provides excellent fire protection
│   │   │   │       │
│   │   │   │       └── SOME MATURE PINES
│   │   │   │           ├── Ponderosa Pine: Develops thick bark with age
│   │   │   │           ├── Jeffrey Pine: Similar thick protective bark
│   │   │   │           └── Both develop thick bark as fire adaptation
│   │   │   │
```
