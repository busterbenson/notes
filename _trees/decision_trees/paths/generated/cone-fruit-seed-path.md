│   │   ├── American Sweetgum (Liquidambar styraciflua)
│   │   │   ├── Oriental Sweetgum (Liquidambar orientalis)
│   │   │   │   ├── Both produce distinctive woody "gumballs" that remain on the tree for months
│   │   │   │   │   ├── Both eventually drop fruits that can cover the ground beneath the tree
│   │   │   │   │   │   ├── Both release tiny seeds from the woody capsules over an extended period
│   ├── CONIFER GENUS GROUP - Winter seed dispersal
│   │   ├── PINE GENUS (Pinus)
│   │   │   ├── Most pine species: Cones mature in fall of second year
│   │   │   │   ├── Seeds released gradually through fall and winter as scales open
│   │   │   │   │   ├── Some species (lodgepole, knobcone) have serotinous cones that remain closed for years
│   │   │   │   │   │   ├── Distinctive woody cones often remain on or under the tree year-round
│   ├── FIR GENUS (Abies)
│   │   ├── Various fir species: Cones mature in fall, disintegrating on the tree
│   │   │   ├── Seeds released as the upright cones break apart scale by scale
│   │   │   │   ├── No intact cones fall to the ground (unlike pines)
│   │   │   │   │   ├── Central cone axis remains standing on branches after seeds disperse
│   ├── SPRUCE GENUS (Picea)
│   │   ├── Various spruce species: Cones mature in fall, opening gradually through winter
│   │   │   ├── Seeds released as scales flex open during dry periods
│   │   │   │   ├── Empty cones often remain on tree for several months
│   │   │   │   │   ├── Distinctive papery textures of spruce cones aid winter identification
