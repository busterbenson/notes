# Flower Path

Look at the flowers of your tree.

```
├── The FLOWERS catch my eye!
│   ├── What COLOR are the flowers?
│   │   ├── WHITE FLOWERS → WHITE FLOWER GROUP
│   │   │   ├── DOGWOOD GENUS (Cornus)
│   │   │   │   ├── Pacific Dogwood: Large white "flowers" (actually bracts)
│   │   │   │   ├── 4-6 white petal-like bracts surrounding true flowers
│   │   │   │   └── Blooms in spring
│   │   │   │
│   │   │   ├── FRUIT TREE GENUS GROUP
│   │   │   │   ├── Cherry: Clusters of white flowers
│   │   │   │   ├── Apple: White to pink blossoms with 5 petals
│   │   │   │   ├── Pear: White flowers in clusters
│   │   │   │   └── All have 5-petaled flowers and produce fleshy fruits
│   │   │   │
│   │   │   └── SERVICEBERRY GENUS (Amelanchier)
│   │   │       ├── Western Serviceberry: Star-like white flowers
│   │   │       ├── 5-petaled flowers appearing early spring
│   │   │       └── Often bloom before leaves fully emerge
│   │   │
│   │   ├── PINK or RED FLOWERS → PINK/RED FLOWER GROUP
│   │   │   ├── REDBUD GENUS (Cercis)
│   │   │   │   ├── Western Redbud: Bright magenta flowers
│   │   │   │   ├── Appear directly on branches before leaves
│   │   │   │   └── Small pea-like flowers in clusters
│   │   │   │
│   │   │   ├── CHERRY/PLUM GENUS (Prunus)
│   │   │   │   ├── Flowering Cherry: Pink to white showy blossoms
│   │   │   │   ├── Flowering Plum: Pink blossoms
│   │   │   │   └── Both have 5-petaled flowers that appear before leaves
│   │   │   │
│   │   │   └── CRABAPPLE GENUS (Malus)
│   │   │       ├── Various Crabapples: Pink to red-pink flowers
│   │   │       ├── 5-petaled flowers in clusters
│   │   │       └── Blooms in spring
│   │   │
│   │   ├── PURPLE or BLUE FLOWERS → PURPLE/BLUE FLOWER GROUP
│   │   │   ├── JACARANDA GENUS (Jacaranda)
│   │   │   │   ├── Jacaranda: Clusters of lavender-blue trumpet flowers
│   │   │   │   ├── Blooms late spring into summer
│   │   │   │   └── Flowers appear before or with feathery leaves
│   │   │   │
│   │   │   ├── PAULOWNIA GENUS (Paulownia)
│   │   │   │   ├── Princess Tree: Lavender trumpet-shaped flowers
│   │   │   │   ├── Bloom in upright clusters before leaves appear
│   │   │   │   └── Flowers resembling foxglove
│   │   │   │
│   │   │   └── CHASTE TREE GENUS (Vitex)
│   │   │       ├── Chaste Tree: Spikes of small lavender-blue flowers
│   │   │       ├── Blooms in summer
│   │   │       └── Flowers resemble small butterfly bushes
│   │   │
│   │   └── YELLOW or GREEN FLOWERS → YELLOW/GREEN FLOWER GROUP
│   │       ├── MAPLE GENUS (Acer)
│   │       │   ├── Various Maples: Small yellowish-green flowers
│   │       │   ├── Appear before or with leaves in spring
│   │       │   └── Not showy but appear in clusters
│   │       │
│   │       ├── OAK GENUS (Quercus)
│   │       │   ├── Various Oaks: Drooping yellowish catkins
│   │       │   ├── Appear with new leaves in spring
│   │       │   └── Male flowers in catkins, females tiny and inconspicuous
│   │       │
│   │       └── CALIFORNIA BUCKEYE (Aesculus californica)
│   │           ├── Showy white to pale yellow flower spires
│   │           ├── Sweet fragrance, attractive to butterflies
│   │           └── Blooms late spring to early summer
│   │
│   ├── How are the flowers ARRANGED?
│   │   ├── CATKINS (hanging chain-like clusters) → CATKIN GROUP
│   │   │   ├── BIRCH GENUS (Betula)
│   │   │   │   ├── Various Birches: Hanging yellowish-brown catkins
│   │   │   │   ├── Male catkins more prominent than female
│   │   │   │   └── Appear in early spring
│   │   │   │
│   │   │   ├── ALDER GENUS (Alnus)
│   │   │   │   ├── Red Alder: Reddish-brown catkins
│   │   │   │   ├── White Alder: Similar catkins
│   │   │   │   └── Often visible through winter
│   │   │   │
│   │   │   └── ASPEN/POPLAR GENUS (Populus)
│   │   │       ├── Quaking Aspen: Long drooping catkins
│   │   │       ├── Cottonwood: Fluffy catkins that release cotton-like seeds
│   │   │       └── Appear before leaves in spring
│   │   │
│   │   ├── CLUSTERS or BUNCHES → CLUSTERED FLOWER GROUP
│   │   │   ├── CHERRY/PLUM GENUS (Prunus)
│   │   │   │   ├── Clustered arrangement of 5-petaled flowers
│   │   │   │   ├── Often appear before leaves
│   │   │   │   └── White to pink colors
│   │   │   │
│   │   │   ├── SERVICEBERRY GENUS (Amelanchier)
│   │   │   │   ├── Star-like white flowers in clusters
│   │   │   │   └── Appear early spring
│   │   │   │
│   │   │   └── MADRONE GENUS (Arbutus)
│   │   │       ├── Pacific Madrone: Clusters of small urn-shaped flowers
│   │   │       ├── White bell-like blossoms in drooping clusters
│   │   │       └── Blooms in spring
│   │   │
│   │   ├── SPIKES or UPRIGHT PLUMES → SPIKE FLOWER GROUP
│   │   │   ├── CALIFORNIA BUCKEYE (Aesculus californica)
│   │   │   │   ├── Cone-shaped upright flower clusters
│   │   │   │   ├── White to pale yellow color
│   │   │   │   └── Very showy in late spring
│   │   │   │
│   │   │   ├── CHESTNUT GENUS (Castanea)
│   │   │   │   ├── Cream-colored catkin-like spikes
│   │   │   │   ├── Strong sweet-spicy scent
│   │   │   │   └── Appear in early summer
│   │   │   │
│   │   │   └── MIMOSA/SILK TREE (Albizia julibrissin)
│   │   │       ├── Fluffy pink pom-pom like flowers
│   │   │       ├── Arranged in upright clusters
│   │   │       └── Blooms in summer
│   │   │
│   │   └── SINGLE LARGE FLOWERS → SHOWY FLOWER GROUP
│   │       ├── DOGWOOD GENUS (Cornus)
│   │       │   ├── Pacific Dogwood: Large white "flowers" (actually bracts)
│   │       │   ├── Usually 4-6 white bracts surrounding true flowers
│   │       │   └── Very distinctive in forest settings
│   │       │
│   │       ├── MAGNOLIA GENUS (Magnolia)
│   │       │   ├── Various Magnolias: Large single flowers
│   │       │   ├── White, pink, or purple colors
│   │       │   └── Often bloom before leaves appear
│   │       │
│   │       └── TULIP TREE (Liriodendron tulipifera)
│   │           ├── Tulip-shaped yellow-green flowers with orange band
│   │           ├── Upright on branch tips
│   │           └── May be hard to see as they're high in the canopy
│   │
│   └── When do the flowers APPEAR?
│       ├── BEFORE LEAVES (on bare branches) → EARLY BLOOM GROUP
│       │   ├── REDBUD GENUS (Cercis)
│       │   │   ├── Western Redbud: Magenta flowers directly on bare branches
│       │   │   └── Very distinctive in early spring
│       │   │
│       │   ├── CHERRY/PLUM GENUS (Prunus)
│       │   │   ├── Flowering Cherry/Plum: Showy blossoms before leaves
│       │   │   └── Creates dramatic flowering display
│       │   │
│       │   └── MAGNOLIA GENUS (Magnolia)
│       │       ├── Star Magnolia: White flowers on bare branches
│       │       ├── Saucer Magnolia: Pink-purple flowers on bare branches
│       │       └── Both bloom before leaf emergence
│       │
│       ├── WITH NEW LEAVES → SIMULTANEOUS BLOOM GROUP
│       │   ├── DOGWOOD GENUS (Cornus)
│       │   │   ├── Pacific Dogwood: White bracts as leaves develop
│       │   │   └── Blooms with spring leaf emergence
│       │   │
│       │   ├── APPLE/CRABAPPLE GENUS (Malus)
│       │   │   ├── Various species: Pink to white flowers with new leaves
│       │   │   └── Classic spring flowering trees
│       │   │
│       │   └── SERVICEBERRY GENUS (Amelanchier)
│       │       ├── Western Serviceberry: White flowers with new leaves
│       │       └── Early spring bloomer
│       │
│       └── AFTER LEAVES FULLY DEVELOPED → LATE BLOOM GROUP
│           ├── CALIFORNIA BUCKEYE (Aesculus californica)
│           │   ├── White flower spikes after leaves fully expanded
│           │   └── Late spring to early summer bloomer
│           │
│           ├── MIMOSA/SILK TREE (Albizia julibrissin)
│           │   ├── Pink fluffy flowers in summer after full leaf development
│           │   └── Provides summer interest when few trees are flowering
│           │
│           └── CRAPE MYRTLE (Lagerstroemia indica)
│               ├── Bright flower clusters in summer
│               ├── Long bloom period (months)
│               └── One of the latest-blooming ornamental trees
```
