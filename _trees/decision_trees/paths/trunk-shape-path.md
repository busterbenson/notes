# Trunk Shape Path

Look at the shape or form of the tree's trunk.

```
├── TRUNK SHAPE or FORM is most distinctive!
│   ├── NUMBER OF MAIN TRUNKS
│   │   ├── SINGLE TRUNK → SINGLE TRUNK TREES
│   │   │   ├── The trunk is perfectly STRAIGHT → STRAIGHT TRUNK GROUP
│   │   │   │   ├── EXTREMELY TALL, UNBRANCHED FOR MOST OF HEIGHT
│   │   │   │   │   ├── REDWOOD & SEQUOIA GROUP (Sequoia, Sequoiadendron, Metasequoia genera)
│   │   │   │   │   │   ├── Coast Redwood (Sequoia sempervirens): Extremely tall with straight, tapering trunk
│   │   │   │   │   │   ├── Giant Sequoia (Sequoiadendron giganteum): Massive straight trunk with buttressed base
│   │   │   │   │   │   └── Dawn Redwood (Metasequoia glyptostroboides): Deciduous with straight central trunk
│   │   │   │   │   │
│   │   │   │   │   ├── TALL PINE SPECIES (Pinus genus)
│   │   │   │   │   │   ├── Ponderosa Pine (P. ponderosa): Tall straight trunk with orange-plated bark
│   │   │   │   │   │   ├── Sugar Pine (P. lambertiana): Extremely tall with straight trunk
│   │   │   │   │   │   ├── Jeffrey Pine (P. jeffreyi): Tall straight trunk with puzzle-piece bark
│   │   │   │   │   │   ├── Western White Pine (P. monticola): Tall straight trunk with smoother bark
│   │   │   │   │   │   ├── Lodgepole Pine (P. contorta): Straight slender trunk in dense stands
│   │   │   │   │   │   └── Other tall pine species with similar characteristics
│   │   │   │   │   │
│   │   │   │   │   ├── TRUE CEDAR GROUP (Cedrus genus)
│   │   │   │   │   │   ├── Deodar Cedar (Cedrus deodara): Tall straight trunk, pyramidal when young
│   │   │   │   │   │   ├── Atlas Cedar (C. atlantica): Straight central trunk with level branches
│   │   │   │   │   │   └── Lebanon Cedar (C. libani): Straight trunk, becoming flat-topped with age
│   │   │   │   │   │
│   │   │   │   │   └── OTHER TALL STRAIGHT-TRUNKED CONIFERS

│   │   │   │   │       ├── Noble Fir (Abies procera): Very straight trunk with regular branch whorls
│   │   │   │   │       ├── Sitka Spruce (Picea sitchensis): Tall straight trunk in mature specimens
│   │   │   │   │       ├── White Fir (Abies concolor): Straight trunk with silvery-gray bark
│   │   │   │   │       ├── Red Fir (Abies magnifica): Straight thick trunk with reddish bark
│   │   │   │   │       ├── Engelmann Spruce (Picea engelmannii): Straight trunk at maturity
│   │   │   │   │       └── Other fir and spruce species with similar form
│   │   │   │   │
│   │   │   │   ├── COLUMNAR or PENCIL-LIKE
│   │   │   │   │   ├── EXTREMELY NARROW PROPORTIONS → PENCIL-FORM GROUP
│   │   │   │   │   │   ├── CYPRESS GENUS (Cupressus)
│   │   │   │   │   │   │   ├── Italian Cypress (Cupressus sempervirens): Classic pencil-like form, very narrow
│   │   │   │   │   │   │   ├── Arizona Cypress (C. arizonica): Can have narrow columnar form
│   │   │   │   │   │   │   └── Monterey Cypress (C. macrocarpa): Sometimes develops narrow form in sheltered areas
│   │   │   │   │   │   │
│   │   │   │   │   │   ├── JUNIPER GENUS (Juniperus)
│   │   │   │   │   │   │   ├── Eastern Redcedar (J. virginiana): Often develops narrow columnar form
│   │   │   │   │   │   │   ├── Rocky Mountain Juniper (J. scopulorum): Sometimes pencil-like in cultivars
│   │   │   │   │   │   │   └── Western Juniper (J. occidentalis): Can form narrow columns in some habitats
│   │   │   │   │   │   │
│   │   │   │   │   │   ├── POPLAR/ASPEN GENUS (Populus)
│   │   │   │   │   │   │   ├── Lombardy Poplar (P. nigra 'Italica'): Classic pencil-form cultivar
│   │   │   │   │   │   │   ├── Other columnar poplar cultivars: Various narrow forms
│   │   │   │   │   │   │   └── Quaking Aspen (P. tremuloides): Can develop columnar form in dense stands
│   │   │   │   │   │   │
│   │   │   │   │   │   └── OTHER PENCIL-FORM TREES
│   │   │   │   │   │       ├── Irish Yew (Taxus baccata 'Fastigiata'): Very narrow columnar form
│   │   │   │   │   │       ├── Columnar European Hornbeam (Carpinus betulus 'Fastigiata'): Narrow upright form
│   │   │   │   │   │       └── Columnar maple cultivars: Various species with narrow cultivated forms
│   │   │   │   │   │
│   │   │   │   │   └── MODERATELY COLUMNAR → COLUMNAR GROUP
│   │   │   │   │       ├── POPLAR/ASPEN GENUS (Populus)
│   │   │   │   │       │   ├── Lombardy Poplar (P. nigra 'Italica'): Distinctive columnar form
│   │   │   │   │       │   ├── Quaking Aspen (P. tremuloides): Straight white trunks in groves
│   │   │   │   │       │   ├── Balsam Poplar (P. balsamifera): Often columnar in forest settings
│   │   │   │   │       │   └── White Poplar (P. alba): Can develop columnar form
│   │   │   │   │       │

│   │   │   │   │       │   └── Other Eucalyptus species with similar growth pattern
│   │   │   │   │       │

│   │   │   │   │           ├── Tulip Tree (Liriodendron tulipifera): Develops columnar trunk in maturity
│   │   │   │   │           ├── Sweet Gum (Liquidambar styraciflua): Often columnar in forest settings
│   │   │   │   │           └── Beech species (Fagus): Sometimes develop columnar form in forest settings
│   │   │   │   │
│   │   │   │   └── SUBSTANTIAL TRUNK DIAMETER → MASSIVE STRAIGHT TRUNK GROUP
│   │   │   │       ├── SEQUOIA GROUP
│   │   │   │       │   ├── Giant Sequoia: World's most massive tree trunks, straight
│   │   │   │       │   ├── Coast Redwood: Huge trunk diameter, very tall
│   │   │   │       │   └── Both have fibrous reddish bark
│   │   │   │       │
│   │   │   │       ├── LARGE FIR SPECIES
│   │   │   │       │   ├── Douglas-fir: Straight, massive trunk in mature trees
│   │   │   │       │   ├── White Fir: Straight substantial trunk
│   │   │   │       │   └── Most have deeply furrowed bark
│   │   │   │       │
│   │   │   │       └── SOME PINE SPECIES
│   │   │   │           ├── Ponderosa Pine: Substantial straight orange-plated trunk
│   │   │   │           ├── Sugar Pine: Massive straight trunk
│   │   │   │           └── Both have distinctive bark patterns
│   │   │   │
│   │   │   ├── The trunk is CURVED, LEANING, or TWISTED → IRREGULAR TRUNK GROUP
│   │   │   │   ├── MODERATELY CURVED OR LEANING → LEANING TRUNK GROUP
│   │   │   │   │   ├── WINDSWEPT CONIFERS

│   │   │   │   │   │   ├── Monterey Pine: Can develop lean in coastal exposure
│   │   │   │   │   │   └── Shore Pine: Often develops lean in coastal winds
│   │   │   │   │   │

│   │   │   │   │

│   │   │   │   │   │

│   │   │   │       │

│   │   │   │           └── California Buckeye: Sometimes with serpentine trunk
│   │   │   │
```

## I'm Not Sure
If you're having trouble identifying the trunk shape:
* **Step back**: Sometimes you need to move farther away to see the overall trunk pattern
* **Look for multiple trunks**: Count how many main trunks grow from the ground
* **Measure with your hands**: If the tree is small enough, you can use your hands to see if the trunk is straight or curved
* **Compare to your body**: Is the trunk straight like when you stand tall, or does it bend and twist?
* **Look for branching height**: Notice if branches start near the ground, in the middle, or only high up

## Tree Detective Tips
★ **Easy** (Good for beginners):
* Counting trunks (single vs. multiple) is the easiest way to start
* Noticing very colorful bark (white, red, orange) is also easy to spot

★★ **Medium** (Takes some practice):
* Determining if a trunk is straight, curved, or twisted
* Identifying the height where major branches begin

★★★ **Challenging** (For experienced tree detectives):
* Distinguishing between different types of multi-trunk patterns
* Identifying specific bark textures and patterns
