│       │   │       ├── All develop pods that turn from green to brown when mature
│       │   │       └── All contain seeds arranged in a row within the pod (key legume trait)
│       │   │
│       │   ├── MEDIUM (1/4 to 1 inch) → MEDIUM FRUIT GROUP
│       │   │   ├── OAK GENUS (Quercus) - Acorns

│       │   │   │   ├── Holly-leaf Cherry (Prunus ilicifolia): Dark fruits with bitter flesh
│       │   │   │   ├── All cherry and plum species produce fleshy drupes with a central stone
│       │   │   │   ├── All develop from a single flower's ovary with a hard pit inside

│       │   │   │
│       │   │   └── MADRONE & MANZANITA GENUS GROUP - Berry-like drupes

│       │   │       ├── Strawberry Tree (Arbutus unedo): Red berry-like fruits (1/2-3/4")
│       │   │       ├── Bigberry Manzanita (Arctostaphylos glauca): Larger berry-like fruits (1/3-1/2")
│       │   │       ├── Common Manzanita (Arctostaphylos manzanita): Small apple-like fruits (1/4-1/3")
│       │   │       ├── All species in this group produce berry-like fruits with mealy flesh
│       │   │       ├── All contain multiple small hard seeds embedded in the flesh
│       │   │       └── All develop a distinctive glossy surface when mature (key trait)

│       │       │
│       │       ├── ALDER GENUS (Alnus) - Tiny seeds in "cones"

│       │           │

│       │           │   ├── Wings permanently attached to aid wind dispersal
│       │           │   └── Extremely small and numerous seeds produced annually
