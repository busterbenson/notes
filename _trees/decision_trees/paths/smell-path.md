# Smell Path

Crush a leaf or needle and smell it.

```
├── The LEAVES or NEEDLES SMELL interesting when crushed!
│   ├── What TYPE of smell do they have?
│   │   ├── PINE-LIKE or RESINOUS → CONIFER SCENT GROUP
│   │   │   ├── What's the SPECIFIC smell like?
│   │   │   │   ├── VANILLA or BUTTERSCOTCH smell → VANILLA-SCENTED BARK GROUP
│   │   │   │   │   ├── PONDEROSA PINE (Pinus ponderosa)
│   │   │   │   │   │   ├── Distinct vanilla/butterscotch smell in bark crevices
│   │   │   │   │   │   ├── Smell strongest on warm, sunny days
│   │   │   │   │   │   └── Yellow-orange puzzle-piece bark when mature
│   │   │   │   │   │
│   │   │   │   │   ├── JEFFREY PINE (Pinus jeffreyi)
│   │   │   │   │   │   ├── Strong vanilla or pineapple scent in bark
│   │   │   │   │   │   ├── Even more pronounced than Ponderosa Pine
│   │   │   │   │   │   └── Key distinguishing feature from Ponderosa
│   │   │   │   │   │
│   │   │   │   │   └── SUGAR PINE (Pinus lambertiana)
│   │   │   │   │       ├── Milder sweet smell in bark
│   │   │   │   │       ├── Extremely long cones (10-20 inches)
│   │   │   │   │       └── Needles in bundles of 5
│   │   │   │   │
│   │   │   │   ├── PENCIL SHAVINGS smell → PENCIL-SCENTED GROUP
│   │   │   │   │   ├── INCENSE-CEDAR (Calocedrus decurrens)
│   │   │   │   │   │   ├── Strong pencil-like smell when foliage crushed
│   │   │   │   │   │   ├── The wood used to make pencils
│   │   │   │   │   │   └── Flat sprays of scale-like leaves
│   │   │   │   │   │
│   │   │   │   │   ├── WESTERN REDCEDAR (Thuja plicata)
│   │   │   │   │   │   ├── Cedar-like smell, similar to pencil shavings
│   │   │   │   │   │   ├── Flat, lacy sprays of scale-like leaves
│   │   │   │   │   │   └── Fibrous reddish-brown bark
│   │   │   │   │   │
│   │   │   │   │   └── OTHER CEDAR-LIKE TREES
│   │   │   │   │       ├── Port Orford-cedar: Similar cedar smell
│   │   │   │   │       ├── Alaska-cedar: Pungent cedar smell
│   │   │   │   │       └── All have flat, scale-like foliage
│   │   │   │   │
│   │   │   │   └── TURPENTINE or RESIN smell → RESINOUS NEEDLE GROUP
│   │   │   │       ├── PINE GENUS (Various Pinus species)
│   │   │   │       │   ├── Most pine needles have resinous smell when crushed
│   │   │   │       │   ├── Smell varies slightly between species
│   │   │   │       │   └── All have needles in bundles of 2-5
│   │   │   │       │
│   │   │   │       ├── FIR GENUS (Abies)
│   │   │   │       │   ├── White Fir: Pleasant, milder resinous scent
│   │   │   │       │   ├── Red Fir: Strong pine-like aroma
│   │   │   │       │   └── All have flat needles, upright cones
│   │   │   │       │
│   │   │   │       └── SPRUCE GENUS (Picea)
│   │   │   │           ├── Sitka Spruce: Strong, pungent smell when crushed
│   │   │   │           ├── Engelmann Spruce: Unpleasant smell when crushed
│   │   │   │           └── All have sharp, stiff needles that grow all around twig
│   │   │   │
│   │   │   ├── Where on the tree is the smell strongest?
│   │   │   │   ├── BARK smells strongest → AROMATIC BARK GROUP
│   │   │   │   │   ├── Ponderosa Pine: Vanilla/butterscotch scent in bark plates
│   │   │   │   │   ├── Jeffrey Pine: Even stronger sweet scent in bark
│   │   │   │   │   └── Incense-cedar: Aromatic bark, especially when wet
│   │   │   │   │
│   │   │   │   ├── NEEDLES or FOLIAGE smell strongest → AROMATIC FOLIAGE GROUP
│   │   │   │   │   ├── Most pines, firs, cedars: Aromatic when crushed
│   │   │   │   │   ├── Redwoods: Sweet, tangy scent when needles crushed
│   │   │   │   │   └── Douglas-fir: Citrusy-pine scent when needles crushed
│   │   │   │   │
│   │   │   │   └── RESIN or SAP smells strongest → AROMATIC RESIN GROUP
│   │   │   │       ├── Pine species: Sticky sap with strong turpentine smell
│   │   │   │       ├── Fir species: Clear, fragrant resin in blisters on young bark
│   │   │   │       └── Spruce species: Hard amber resin with pungent smell
│   │   │   │
│   │   │   └── How STRONG is the smell?
│   │   │       ├── VERY STRONG, detectable from a distance → STRONGLY AROMATIC GROUP
│   │   │       │   ├── Jeffrey Pine: Can sometimes smell bark on warm days
│   │   │       │   ├── Incense-cedar: Strong aroma, especially after rain
│   │   │       │   └── Port Orford-cedar: Very strong when foliage crushed
│   │   │       │
│   │   │       ├── MODERATE, noticeable when close → MODERATELY AROMATIC GROUP
│   │   │       │   ├── Most pines: Need to crush needles to detect smell
│   │   │       │   ├── Fir species: Moderate fragrance when needles broken
│   │   │       │   └── Redwood species: Gentle aroma when foliage crushed
│   │   │       │
│   │   │       └── SUBTLE, need to crush and smell closely → MILDLY AROMATIC GROUP
│   │   │           ├── Hemlock species: Mild aroma when crushed
│   │   │           ├── Some young pines: Subtle fragrance
│   │   │           └── Western white pine: Milder scent than other pines
│   │   │
│   │   ├── MINTY or EUCALYPTUS-like → MINT SCENT GROUP
│   │   │   ├── How STRONG is the minty smell?
│   │   │   │   ├── VERY STRONG, menthol-like → STRONG MINT GROUP
│   │   │   │   │   ├── BLUE GUM EUCALYPTUS (Eucalyptus globulus)
│   │   │   │   │   │   ├── Powerful menthol or cough drop smell
│   │   │   │   │   │   ├── Blue-gray juvenile leaves, different from adult leaves
│   │   │   │   │   │   └── Peeling bark in multiple colors
│   │   │   │   │   │
│   │   │   │   │   ├── RIVER RED GUM (Eucalyptus camaldulensis)
│   │   │   │   │   │   ├── Strong eucalyptus smell when leaves crushed
│   │   │   │   │   │   ├── Narrow, lance-shaped leaves that hang vertically
│   │   │   │   │   │   └── Smooth bark with patches of red, white and gray
│   │   │   │   │   │
│   │   │   │   │   └── PEPPERMINT EUCALYPTUS (Eucalyptus piperita)
│   │   │   │   │       ├── Distinct peppermint smell when crushed
│   │   │   │   │       ├── Planted as ornamental in some areas
│   │   │   │   │       └── Similar leaf shape to Blue Gum
│   │   │   │   │
│   │   │   │   ├── MODERATE, less medicinal → MODERATE MINT GROUP
│   │   │   │   │   ├── SILVER DOLLAR GUM (Eucalyptus polyanthemos)
│   │   │   │   │   │   ├── Milder eucalyptus smell than other species
│   │   │   │   │   │   ├── Round, silver-blue leaves resembling coins
│   │   │   │   │   │   └── Leaves keep round shape even when mature
│   │   │   │   │   │
│   │   │   │   │   ├── LEMON-SCENTED GUM (Corymbia citriodora)
│   │   │   │   │   │   ├── Minty-citrus smell when leaves crushed
│   │   │   │   │   │   ├── Very tall with smooth, powdery white bark
│   │   │   │   │   │   └── Narrow, lance-shaped leaves
│   │   │   │   │   │
│   │   │   │   │   └── RED FLOWERING GUM (Corymbia ficifolia)
│   │   │   │   │       ├── Mild eucalyptus smell when leaves crushed
│   │   │   │   │       ├── Broader leaves than many eucalyptus
│   │   │   │   │       └── Known for spectacular red/orange flowers
│   │   │   │   │
│   │   │   │   └── SUBTLE, mixed with other scents → MILD MINT GROUP
│   │   │   │       ├── SOME BAY TREES
│   │   │   │       │   ├── California Bay: Can have minty notes mixed with bay scent
│   │   │   │       │   └── Stronger bay spice smell overall
│   │   │   │       │
│   │   │   │       ├── CAMPHOR TREE (Cinnamomum camphora)
│   │   │   │       │   ├── Minty-medicinal smell with camphor notes
│   │   │   │       │   ├── Glossy, waxy leaves
│   │   │   │       │   └── Planted as ornamental shade tree
│   │   │   │       │
│   │   │   │       └── SOME ORNAMENTAL EUCALYPTUS
│   │   │   │           ├── Various species with milder eucalyptus scent
│   │   │   │           ├── Often selected for landscaping for less intense smell
│   │   │   │           └── Varied leaf shapes and bark patterns
│   │   │   │
│   │   │   ├── Is the minty smell MIXED with other scents?
│   │   │   │   ├── MINT + CITRUS notes → MINT-CITRUS GROUP
│   │   │   │   │   ├── LEMON-SCENTED GUM (Corymbia citriodora)
│   │   │   │   │   │   ├── Strong lemon-mint smell when crushed
│   │   │   │   │   │   ├── Long, narrow leaves
│   │   │   │   │   │   └── Smooth, pale bark
│   │   │   │   │   │
│   │   │   │   │   ├── LEMON EUCALYPTUS (Eucalyptus citriodora)
│   │   │   │   │   │   ├── Similar lemon-mint fragrance
│   │   │   │   │   │   ├── Sometimes used for essential oils
│   │   │   │   │   │   └── Smooth, powdery white bark
│   │   │   │   │   │
│   │   │   │   │   └── SOME MYRTLE SPECIES
│   │   │   │   │       ├── Various myrtles can have mint-citrus notes
│   │   │   │   │       ├── Generally smaller than eucalyptus
│   │   │   │   │       └── Often planted as ornamentals
│   │   │   │   │
│   │   │   │   ├── MINT + MEDICINAL notes → MEDICINAL GROUP
│   │   │   │   │   ├── BLUE GUM EUCALYPTUS (Eucalyptus globulus)
│   │   │   │   │   │   ├── Mint with strong medicinal overtones
│   │   │   │   │   │   ├── The classic "eucalyptus" smell

│   │   │   │   │   │
│   │   │   │   │   ├── CAMPHOR TREE (Cinnamomum camphora)
│   │   │   │   │   │   ├── Medicinal-mint smell with camphor notes
│   │   │   │   │   │   ├── Source of camphor in traditional medicine
│   │   │   │   │   │   └── Glossy evergreen leaves
│   │   │   │   │   │
│   │   │   │   │   └── VARIOUS MEDICINAL PLANTS
│   │   │   │   │       ├── Tea Tree: Sometimes planted in warm areas
│   │   │   │   │       ├── Australian Melaleuca species
│   │   │   │   │       └── All have strong medicinal properties
│   │   │   │   │
│   │   │   │   └── MINT + WOODY notes → WOODY-MINT GROUP

│   │   │   │       │   ├── Many have woody undertones with mint smell
```
