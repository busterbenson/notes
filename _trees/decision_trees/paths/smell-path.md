# Smell Path

Crush a leaf or needle and smell it.

**Special Feature Pages:**
- [Eucalyptus Group](groups/eucalyptus-group.md) - Learn about the aromatic eucalyptus species
- [Pine Group](groups/pine-group.md) - Discover how to identify pines by their distinctive scents

```
├── The LEAVES or NEEDLES SMELL interesting when crushed!
│   ├── What TYPE of smell do they have?
│   │   ├── I'M NOT SURE about the smell type → SMELL HELPER
│   │   │   ├── That's okay! Here's how to figure it out:
│   │   │   │   ├── Try crushing the leaf or needle between your fingers
│   │   │   │   ├── Take small sniffs (don't breathe in too deeply!)
│   │   │   │   ├── Think about familiar smells like Christmas trees, mint gum, or cooking spices
│   │   │   │   └── Ask an adult to help describe what they smell
│   │   │   │
│   │   │   ├── Common smell categories for kids:
│   │   │   │   ├── Christmas tree smell → Go to PINE-LIKE
│   │   │   │   ├── Mint gum or cough drop smell → Go to MINTY
│   │   │   │   ├── Lemon or orange smell → Go to CITRUSY
│   │   │   │   ├── Cooking spice smell → Go to BAY LEAF
│   │   │   │   ├── Nice sweet smell → Go to SWEET
│   │   │   │   └── Unusual or strong smell → Go to OTHER
│   │   │   │
│   │   │   └── Still not sure? That's perfectly fine!
│   │   │       ├── Try looking at other features instead
│   │   │       ├── Check the tree's leaves, bark, or shape
│   │   │       └── You can always come back to smell later
│   │   │
│   │   ├── PINE-LIKE or RESINOUS → CONIFER SCENT GROUP
│   │   │   ├── What's the SPECIFIC smell like?
│   │   │   │   ├── VANILLA or BUTTERSCOTCH smell → VANILLA-SCENTED BARK GROUP
│   │   │   │   │   ├── PINE GENUS (Pinus) - Vanilla-scented species
│   │   │   │   │   │   ├── Ponderosa Pine (Pinus ponderosa): Distinct vanilla/butterscotch smell in bark crevices
│   │   │   │   │   │   ├── Jeffrey Pine (Pinus jeffreyi): Strong vanilla or pineapple scent, more pronounced than Ponderosa
│   │   │   │   │   │   ├── Sugar Pine (Pinus lambertiana): Milder sweet smell in bark, extremely long cones (10-20 inches)
│   │   │   │   │   │   ├── Western Yellow Pine: Similar vanilla scent to Ponderosa (considered a variety by some)
│   │   │   │   │   │   ├── Coulter Pine: Can have subtle vanilla notes in bark crevices
│   │   │   │   │   │   ├── These "yellow pines" all have vanilla-scented bark that's strongest on warm, sunny days
│   │   │   │   │   │   ├── All develop characteristic yellow-orange "puzzle piece" bark pattern when mature
│   │   │   │   │   │   └── All pine species have needles in bundles and resin canals (key pine trait)
│   │   │   │   │
│   │   │   │   ├── PENCIL SHAVINGS smell → PENCIL-SCENTED GROUP
│   │   │   │   │   ├── INCENSE-CEDAR GENUS (Calocedrus)
│   │   │   │   │   │   ├── Incense-cedar (Calocedrus decurrens): Strong pencil-like smell when foliage crushed
│   │   │   │   │   │   ├── Formosan Incense-cedar: Similar aromatic qualities, less common in cultivation
│   │   │   │   │   │   ├── Chinese Incense-cedar: Related species with similar scent profile
│   │   │   │   │   │   ├── This genus provides the wood traditionally used to make pencils
│   │   │   │   │   │   ├── All Calocedrus species have strong aromatic compounds in their wood and foliage
│   │   │   │   │   │   └── All have flat sprays of scale-like leaves arranged in distinct pattern (key incense-cedar trait)
│   │   │   │   │   │
│   │   │   │   │   ├── REDCEDAR GENUS (Thuja)
│   │   │   │   │   │   ├── Western Redcedar (Thuja plicata): Cedar-like smell, similar to pencil shavings
│   │   │   │   │   │   ├── Eastern Redcedar: Aromatic wood used for cedar chests and closets
│   │   │   │   │   │   ├── Giant Arborvitae: Cedar smell when foliage crushed
│   │   │   │   │   │   ├── Oriental Thuja: Similar aromatic properties in landscaping
│   │   │   │   │   │   ├── All Thuja species contain aromatic oils that give the characteristic cedar smell
│   │   │   │   │   │   ├── All have flat, lacy sprays of scale-like leaves that overlap like shingles
│   │   │   │   │   │   └── All develop fibrous, reddish-brown bark with age (key redcedar trait)
│   │   │   │   │   │
│   │   │   │   │   └── OTHER AROMATIC CEDAR GENUS (Chamaecyparis)
│   │   │   │   │       ├── Port Orford-cedar (Chamaecyparis lawsoniana): Sweet cedar smell in foliage and wood
│   │   │   │   │       ├── Alaska-cedar (Chamaecyparis nootkatensis): Pungent cedar smell when crushed
│   │   │   │   │       ├── Hinoki Cypress: Sweet resinous aroma used in Japanese incense
│   │   │   │   │       ├── Sawara Cypress: Similar sweet cedar aroma
│   │   │   │   │       ├── All Chamaecyparis species produce aromatic compounds in their wood and foliage
│   │   │   │   │       └── All have flat, scale-like foliage and small rounded cones (key false-cedar trait)
│   │   │   │   │
│   │   │   │   └── TURPENTINE or RESIN smell → RESINOUS NEEDLE GROUP
│   │   │   │       ├── PINE GENUS (Pinus) - Resinous species
│   │   │   │       │   ├── Lodgepole Pine (Pinus contorta): Strong resinous smell when needles crushed
│   │   │   │       │   ├── Monterey Pine (Pinus radiata): Distinctive turpentine-like aroma in needles
│   │   │   │       │   ├── Gray Pine (Pinus sabiniana): Strongly resinous foliage smell
│   │   │   │       │   ├── Bishop Pine (Pinus muricata): Sharp resinous scent in needles
│   │   │   │       │   ├── Knobcone Pine (Pinus attenuata): Turpentine-like smell when crushed
│   │   │   │       │   ├── All pine species have resinous needles with distinctive smell when crushed
│   │   │   │       │   ├── The smell varies slightly between species but all have turpentine/resin base notes
│   │   │   │       │   └── All have needles in bundles of 2-5 with resin canals (key pine trait)
│   │   │   │       │
│   │   │   │       ├── FIR GENUS (Abies)
│   │   │   │       │   ├── White Fir (Abies concolor): Pleasant, milder resinous scent in needles
│   │   │   │       │   ├── Red Fir (Abies magnifica): Strong pine-like aroma when crushed
│   │   │   │       │   ├── Grand Fir (Abies grandis): Sweet citrusy-resinous scent when crushed
│   │   │   │       │   ├── California Red Fir (Abies magnifica var. shastensis): Strong resinous aroma
│   │   │   │       │   ├── Noble Fir (Abies procera): Mild but distinctive resinous smell
│   │   │   │       │   ├── All fir species produce aromatic compounds in their needles
│   │   │   │       │   ├── All have distinctive resin blisters on young bark that are strongly aromatic
│   │   │   │       │   └── All have flat needles and upright cones that disintegrate on the tree (key fir trait)
│   │   │   │       │
│   │   │   │       └── SPRUCE GENUS (Picea)
│   │   │   │           ├── Sitka Spruce (Picea sitchensis): Strong, pungent smell when needles crushed
│   │   │   │           ├── Engelmann Spruce (Picea engelmannii): Unpleasant "cat urine" smell when crushed
│   │   │   │           ├── Blue Spruce (Picea pungens): Sharp resinous smell in foliage
│   │   │   │           ├── Norway Spruce (Picea abies): Distinctive strong resinous aroma
│   │   │   │           ├── Brewer Spruce (Picea breweriana): Milder but still resinous scent
│   │   │   │           ├── All spruce species contain aromatic compounds in their needles
│   │   │   │           ├── Each species has a slightly different scent profile (some more pleasant than others)
│   │   │   │           └── All have sharp, stiff needles that grow all around twig and hang down (key spruce trait)
│   │   │   │
│   │   │   ├── Where on the tree is the smell strongest?
│   │   │   │   ├── BARK smells strongest → AROMATIC BARK GROUP
│   │   │   │   │   ├── PINE GENUS (Pinus) - Aromatic bark species
│   │   │   │   │   │   ├── Ponderosa Pine: Vanilla/butterscotch scent in bark plates
│   │   │   │   │   │   ├── Jeffrey Pine: Even stronger sweet scent in bark
│   │   │   │   │   │   ├── Sugar Pine: Milder sweet scent in bark plates
│   │   │   │   │   │   ├── Western Yellow Pine: Similar to Ponderosa Pine
│   │   │   │   │   │   ├── The "yellow pine" group all have aromatic compounds in their bark
│   │   │   │   │   │   └── All develop characteristic puzzle-piece bark that's most fragrant on warm days
│   │   │   │   │   │
│   │   │   │   │   ├── CEDAR GENUS GROUP - Aromatic bark species
│   │   │   │   │   │   ├── Incense-cedar: Aromatic bark, especially when wet
│   │   │   │   │   │   ├── Western Redcedar: Fibrous bark with cedar smell
│   │   │   │   │   │   ├── Port Orford-cedar: Aromatic bark with sweet resinous scent
│   │   │   │   │   │   ├── Alaska-cedar: Pungent cedar-like smell in bark
│   │   │   │   │   │   └── All cedar-like species contain aromatic compounds throughout bark and wood
│   │   │   │   │   │
│   │   │   │   │   └── OTHER AROMATIC BARK SPECIES
│   │   │   │   │       ├── California Bay Laurel: Strong bay scent in bark and wood
│   │   │   │   │       ├── Camphor Tree: Medicinal camphor smell in bark
│   │   │   │   │       ├── Sweet Birch: Wintergreen scent in bark
│   │   │   │   │       └── All these species have aromatic compounds throughout their wood
│   │   │   │   │
│   │   │   │   ├── NEEDLES or FOLIAGE smell strongest → AROMATIC FOLIAGE GROUP
│   │   │   │   │   ├── PINE, FIR, SPRUCE GENERA
│   │   │   │   │   │   ├── Most pine species: Strongly resinous needle scent when crushed
│   │   │   │   │   │   ├── Fir species: Various pleasant resinous and sometimes citrusy scents
│   │   │   │   │   │   ├── Spruce species: Sharp, distinctive aromas in needles
│   │   │   │   │   │   └── All conifers have aromatic compounds in their needles/foliage
│   │   │   │   │   │
│   │   │   │   │   ├── REDWOOD GENUS GROUP
│   │   │   │   │   │   ├── Coast Redwood (Sequoia sempervirens): Sweet, tangy scent when needles crushed
│   │   │   │   │   │   ├── Giant Sequoia (Sequoiadendron giganteum): Similar but distinctive aroma
│   │   │   │   │   │   ├── Dawn Redwood (Metasequoia): Mild sweet smell in foliage

│   │   │   │   │       ├── Bigcone Douglas-fir: Strong resinous needle scent
│   │   │   │   │       ├── All Douglas-fir species have distinctive citrusy notes in their needle aroma
│   │   │   │   │       └── All have flat needles with white bands underneath (key Douglas-fir trait)
│   │   │   │   │
│   │   │   │   └── RESIN or SAP smells strongest → AROMATIC RESIN GROUP
│   │   │   │       ├── PINE GENUS (Pinus) - Resinous species
│   │   │   │       │   ├── Most pine species: Sticky sap with strong turpentine smell
│   │   │   │       │   ├── Gray Pine: Particularly resinous with strong-smelling sap
│   │   │   │       │   ├── Monterey Pine: Copious aromatic resin when injured
│   │   │   │       │   ├── Torrey Pine: Strong-smelling resin when bark is damaged
│   │   │   │       │   └── All pines produce aromatic resin when injured (key pine trait)
│   │   │   │       │
│   │   │   │       ├── FIR GENUS (Abies)
│   │   │   │       │   ├── White Fir: Clear, fragrant resin in blisters on young bark
│   │   │   │       │   ├── Grand Fir: Aromatic resin blisters
│   │   │   │       │   ├── Red Fir: Resin with distinctive pine-like smell
│   │   │   │       │   ├── All fir species have resin blisters in young bark
│   │   │   │       │   └── All produce clear aromatic resin when injured (key fir trait)

│   │   │   │       └── SPRUCE GENUS (Picea)

│   │   │   │           ├── Blue Spruce: Aromatic resin when bark is damaged

│   │   │   │           └── All have harder, more crystalline resin than pines and firs (key spruce trait)

│   │   │       │       └── All these species produce powerful aromatic compounds
│   │   │       │
│   │   │       ├── MODERATE, noticeable when close → MODERATELY AROMATIC GROUP
│   │   │       │   ├── PINE GENUS (Pinus) - Moderate aromatics
```
